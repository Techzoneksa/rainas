# Raina Prototype

هذا المجلد يحتوي نسخة العرض المرجعية لتطبيق Raina — رأينا.

## الغرض

- مرجع للتصميم والتدفقات الحالية.
- مرجع للـ Demo Data والنسخة المستقلة.
- لا يمثل Production Architecture.

## التشغيل

النسخة الأسهل للعرض:

```text
prototype/dist/raina-standalone.html
```

أو من داخل هذا المجلد:

```bash
npm.cmd run build
npm.cmd run dev
```
